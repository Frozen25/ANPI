Descripción detallada

template<typename T>
class anpi::Plot2d< T >

Two-dimensional plots

Given a set of x coordinates and a corresponding set of y values, plot a line between each point (x,y).

You give a pair of vectors with the x and y values with the plot() method to plot a curve. You may overlay as many curves as you need simply by calling plot() as many times as you need to.

Finally, you call show() to display the window with all plotted curves.
